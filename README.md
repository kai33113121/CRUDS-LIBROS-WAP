# CRUDS-LIBROS-WAP

Realizar la codificación del crud de su proyecto para una de las tablas
maestras.

Involucre los elementos de orden MVC, validación de usuario, menú y cierre de
sesión vistos en este instructivo.

Para presentar la evidencia se debe adjuntar, la carpeta del código desarrollado
y un archivo en .txt con el código SQL para crear la base de datos, además del
línk de un pequeño vídeo donde se muestre su funcionamiento.

Fecha límite de entrega: Jueves.
